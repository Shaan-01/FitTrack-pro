# FitTrack Pro – Daily Fitness Tracker

## 📌 Project Overview

FitTrack Pro is a responsive web-based fitness tracker developed using **HTML, CSS, and JavaScript**. The application helps users plan and track their daily workouts, monitor their fitness progress, and maintain consistency through features such as workout planning, BMI calculation, progress tracking, and streak management. All user data is stored using **Local Storage**, allowing the application to work without a backend database.

---

## 🎯 Objectives

* Help users organize their daily workout routines.
* Track workout completion and daily progress.
* Calculate and display Body Mass Index (BMI).
* Encourage consistency through daily streaks and achievements.
* Provide a modern, responsive, and user-friendly interface.

---

## ✨ Features

### 🏋️ Workout Planner

* Add, edit, and delete exercises.
* Organize workouts for each day of the week.
* Store exercise details such as:

  * Exercise Name
  * Sets
  * Repetitions (Reps)
  * Duration (Optional)

### 📊 Progress Tracking

* Daily workout progress bar.
* Percentage completion.
* Workout completion message after finishing all exercises.

### 🔥 Daily Streak Counter

* Tracks consecutive workout days.
* Resets streak if a workout is missed.
* Displays current and longest streak.

### ⚖️ BMI Calculator

* Calculates BMI using height and weight.
* Displays BMI category:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese

### 🌙 Theme Support

* Light Mode
* Dark Mode
* Theme preference saved using Local Storage.

### 🏆 Achievements

* Unlock badges for workout milestones.
* Motivate users to maintain consistency.

### 📈 Statistics Dashboard

* Total exercises completed.
* Total workouts completed.
* Weekly and monthly progress summary.
* Estimated calories burned.

### 💧 Additional Features

* Water intake tracker.
* Workout timer.
* Motivational fitness quotes.
* Responsive design for desktop and mobile devices.

 How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your web browser (or use the Live Server extension).
4. Start adding workouts and tracking your fitness progress.
